# prienz-website

프렌즈(prienz) 기업 소개 정적 웹사이트

Light GSD 채택: ✅

## 스택

- HTML + CSS + JS (프레임워크 없음)
- GitHub Pages 정적 호스팅
- 커스텀 도메인: prienz.co.kr

## 구조

```
/
├── index.html
├── css/
├── js/
├── assets/
└── CNAME
```

## 배포

- GitHub Pages (main 브랜치)
- CNAME 파일로 커스텀 도메인 연결

## 개발

- 로컬 확인: 브라우저에서 index.html 직접 열기 또는 `python3 -m http.server 8000`
